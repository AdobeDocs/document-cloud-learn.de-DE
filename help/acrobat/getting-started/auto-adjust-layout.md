Erfahren Sie mehr über den neuen Bearbeitungsmodus, mit dem Inhalte umfließen und das Layout auf Seiten Ihres PDF automatisch angepasst wird. In diesem Video-Tutorial wird das [neue Acrobat-Erlebnis](new-experience.md) verwendet.

>Das Layout für die automatische Anpassung unterstützt noch nicht alle PDF-Inhalte oder -Sprachen.
