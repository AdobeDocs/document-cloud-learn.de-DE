# Unterschriften einholen.

Sorgen Sie für konstante Prozesse, indem Sie rechtsverbindliche [E-Signaturen](https://www.adobe.com/de/acrobat/online/request-signature.html){target="_blank"} von anderen einholen - überall.

>[!IMPORTANT]
>
>Sie müssen zusätzlich zu Ihrem Acrobat Enterprise-Abonnement über eine Acrobat Sign Enterprise-Berechtigung verfügen.

>[!VIDEO](https://video.tv.adobe.com/v/3409540?captions=ger&quality=12&learn=on&hidetitle=true)

>[!TIP]
>
>Erfahren Sie, wie Sie [Ihre Dokumente](track.md), die zur Signatur gesendet wurden, an einem einfachen Ort verfolgen können.
