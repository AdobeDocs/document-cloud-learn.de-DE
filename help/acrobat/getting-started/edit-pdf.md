# Bearbeiten von Text auf einem PDF

Lernen Sie, wie Sie auf Ihrem PDF Wörter, Sätze oder sogar ganze Seiten bearbeiten können. Wenn du Text direkt in Acrobat bearbeitest, kannst du Inhalte präzise und professionell aktualisieren, ohne zwischen den Programmen hin- und herwechseln zu müssen. In diesem Video-Tutorial wird das [neue Acrobat-Erlebnis](new-experience.md) verwendet.
