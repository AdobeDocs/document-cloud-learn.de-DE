# Wann eignet sich die Option Kopie erstellen?

Eine Vereinbarung wurde bereits gesendet. Sie ist jedoch abgelaufen, wurde abgebrochen oder Sie müssen sie erneut senden? Mit der Funktion zum Erstellen einer Kopie können Sie eine vorhandene Vereinbarung duplizieren, Änderungen an den Anforderungen anpassen und innerhalb von Sekunden erneut versenden. Keine Wiederherstellung von Grund auf. Das Erstellen einer Kopie ist in folgenden Fällen verfügbar:

* Sie sind der ursprüngliche Absender der Vereinbarung.
* Die Vereinbarung wurde vom modernen Erlebnis &quot;Signatur anfordern&quot; gesendet.
* Die Vereinbarung enthält keine Rolle als Zeugenempfänger.
* Die Vereinbarung wurde aufgrund einer Dokumentenaufbewahrung oder eines Löschens der DSGVO nicht gelöscht.
* Die Vereinbarung erfordert keinen Zugriff auf eine Bibliotheksvorlage, auf die der Absender keinen Zugriff mehr hat.

>[!NOTE]
>
>Das Erstellen einer Kopie ist für Abos auf Enterprise-Ebene verfügbar.
