# Erforderliche Auswahlen durch Kontrollkästchengruppen führen

Erfahren Sie, wie Sie Kontrollkästchengruppen verwenden, um zu steuern, wie viele Optionen ein Unterzeichner in einem Formular auswählen kann. Kontrollkästchengruppen helfen, sicherzustellen, dass Unterzeichner Dokumente korrekt vervollständigen, indem einfache Regeln durchgesetzt werden, z. B. eine minimale oder maximale Anzahl von Auswahlen, bevor eine Vereinbarung gesendet werden kann.

Erfahren Sie, wie Kontrollkästchengruppen:

* Unvollständige oder ungültige Einreichungen verhindern
* Unterzeichner mit klarem, eingebundenem Feedback unterstützen
* Verbesserung der Genauigkeit der in Formularen erfassten Informationen
